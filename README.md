# Skills EstadoRed

**Herramientas descargables para lectura crítica, auditoría y trabajo con IA.**

Skills EstadoRed es una colección de recursos abiertos para usar inteligencia artificial con método: leer mejor, auditar mejor y construir con más criterio.

Estas skills no buscan sustituir el juicio humano. Funcionan como estructuras de lectura, revisión y trabajo: ordenan preguntas, hacen visibles los vacíos, separan hechos de interpretaciones y ayudan a documentar decisiones.

## Filosofía

En EstadoRed usamos IA para pensamiento crítico.

Eso significa que una herramienta de IA no debe usarse solo para producir más rápido, sino también para preguntar mejor, revisar con más cuidado y hacer explícitos los criterios que normalmente quedan ocultos en un flujo de trabajo.

Las skills de este repositorio están pensadas como pequeños paquetes de método: instrucciones, ejemplos y checklists que pueden adaptarse a tareas editoriales, educativas, técnicas o de investigación.

## Skills disponibles

### Skill 01 — Leer en red

Una skill para analizar noticias, comunicados, leyes, reportes, anuncios empresariales, publicaciones virales o narrativas públicas mediante seis preguntas:

1. ¿Qué pasó realmente?
2. ¿Quién lo dice?
3. ¿Qué falta saber?
4. ¿Quién gana si lo creemos así?
5. ¿Qué es certeza y qué es interpretación?
6. ¿Qué conviene esperar antes de concluir?

Directorio:

```txt
leer-en-red/
```

## Estructura del repositorio

```txt
skills-estadored/
  README.md
  LICENSE
  leer-en-red/
    SKILL.md
    README.md
    ejemplos.md
```

Cada skill puede incluir:

* `SKILL.md`: instrucciones principales para la IA.
* `README.md`: explicación para personas usuarias.
* `ejemplos.md`: ejemplos de aplicación.
* `checklist.md`: lista de revisión, cuando aplique.
* `references/`: materiales de apoyo, cuando aplique.
* `scripts/`: scripts auxiliares, cuando aplique.

## Cómo usar una skill

1. Entra al directorio de la skill que quieras usar.
2. Lee el archivo `README.md`.
3. Copia o adapta el contenido de `SKILL.md`.
4. Úsalo como instrucción en ChatGPT, Codex u otro entorno compatible.
5. Ajusta la skill según tu caso, contexto o flujo de trabajo.

## Principios de uso

Las skills de EstadoRed están diseñadas para:

* apoyar la lectura crítica;
* fortalecer la trazabilidad;
* ordenar procesos de revisión;
* separar hechos, inferencias y especulación;
* hacer visibles los límites de una respuesta;
* promover el uso responsable de IA en tareas editoriales, técnicas y de investigación.

No sustituyen investigación, edición, verificación, asesoría profesional ni revisión humana.
